# Intelligent Demand Forecasting

A retail demand forecasting project built to demonstrate statistical analysis, time series forecasting, and (eventually) machine learning skills, grounded in my professional background as a data forecast analyst.

**Dataset:** Rossmann Store Sales (Kaggle)

**Background:** I currently work as a data forecast analyst and am pursuing an M.S. in Data Analytics at Penn State. This repo is where I'm applying coursework to a real, self-directed forecasting problem as I build toward a data science / AI engineering career.

**Status:** Exploratory data analysis in progress.

## Structure

- `data/raw/` — original, unmodified source data
- `data/processed/` — cleaned data ready for analysis
- `analysis/` — R Markdown notebooks (business understanding, EDA, modeling)
- `docs/project_journal.md` — running log of work sessions and findings
